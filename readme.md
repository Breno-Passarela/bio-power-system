# Bio Power (EN)

Bio Power is a web platform for buying supplements, vitamins, and wellness products with a clean, guided shopping experience. The goal is to provide clarity, trust, and convenience without heavy front-end frameworks.

## Highlights

- Catalog of supplements, vitamins, and wellness products
- Clear navigation focused on usability
- Responsive layout in progress for desktop and mobile
- Minimal stack (HTML, CSS, JS, C backend) for full code control

## Tech Stack

[![Tech stack icons](https://skillicons.dev/icons?i=js,html,css,c,bootstrap)](https://skillicons.dev)

## Project Structure

- website/: frontend assets (pages, CSS, JS, images)
- back-end/: C sources

## Getting Started

1. Clone the repo

```
git clone https://github.com/<your-user>/bio-power-system.git
cd bio-power-system
```

2. Preview the frontend

- Open website/home.html directly, or serve the folder:

```
cd website
python -m http.server 8000
```

3. Backend (C)

- Sources are in back-end/. Add your toolchain commands or Makefile as needed.

## Contributing

Issues and pull requests are welcome. Please keep contributions small, focused, and well-described.
