# Bio Power {#bio-power}

Bio Power is a web platform for buying supplements, vitamins, and wellness products with a clean and guided shopping experience. The goal is to deliver clarity, trust, and convenience without relying on heavy front-end frameworks.

## Key Features

- Full catalog of supplements, vitamins, and wellness products
- Simple, clear navigation with focus on usability
- Responsive design in progress for multiple devices
- Minimal tech stack (HTML, CSS, JS, C backend) for full control over the codebase

## Tech Stack

[![Tech stack icons](https://skillicons.dev/icons?i=js,html,css,c,bootstrap)](https://skillicons.dev)

## Project Structure (high level)

- website/: frontend assets (pages, CSS, JS, images)
- back-end/: C sources

## Getting Started

1. Clone

```
git clone https://github.com/<your-user>/bio-power-system.git
cd bio-power-system
```

2. Frontend preview

Open website/home.html directly in the browser or serve the folder with any static server (for example: python -m http.server 8000 inside website/).

3. Backend (C)

The backend sources live in back-end/. Build/run steps depend on your environment; if you have a makefile or specific commands, add them here.

Issues and pull requests are welcome. Please keep contributions small and focused.
